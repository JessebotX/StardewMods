Features

  • Allows the player to sprint (Hold/Toggle)
  • Stamina drain option for when you are sprinting
  • Prevent player from sprinting when over-exerted
  • All features highly configurable
    • See configuration documentation

Usage

  • When in game (your farmer can move), hold down the configured button (default Left Shift)

Install

Dependencies

  • SMAPI [REQUIRED] - the Stardew Valley Mod Loader
  • Generic Mod Config Menu [OPTIONAL] - Allows editing configuration of this mod in game

Installation

  1. Install SMAPI
  2. [OPTIONAL] Install Generic Mod Config Menu (similar install steps to this mod)
  3. Install this mod and extract/unzip it into your Stardew Valley/Mods folder

need help? check out Player's Guide to Installing Mods or visit the Stardew Valley Discord

Uninstallation

  • Delete the folder Sprint Sprint from your Stardew Valley/Mods folder

Other

  • Controller supported, just check out the configuration documentation
  • Works in singleplayer and multiplayer (local changes; only people who have the mod install works)
  • Tested on Windows 10 with Stardew Valley 1.4.5 & SMAPI 3.6.1(Sprint Sprint v2.0.0)

Configuration

See configuration documentation for all options There are 2 ways to edit the config file
  • Download optional dependency Generic Mod Config Menu and you can configure the mod to your liking while in the title screen
    on a much friendlier user interface (A button is added to the title screen in the bottom left to access the config menus.)
  • Run the game with this mod installed once to generate a config.json in your Stardew Valley/Mods/Sprint Sprint/
    folder and directly edit the config.json file with a text editor (i.e. Notepad, Notepad++, Sublime Text, etc.)
If you choose the second option, you NEED to run the game once in order for a config.json to appear.
If the config.json gets replaced/deleted, the game will generate a new config.json with default settings
